EQUIPMENT SYSTEM

Equipment System Overview

Understand Star Sailors' gear mechanics to lay a solid foundation for your build path

Equipment System Introduction

Star Sailors' equipment system is the core pathway to boosting character combat power. Each companion can equip three gear types -- Weapon, Armor, and Accessory -- with different rarities offering varying stat bonuses and set effects. Gear is obtained through dungeon drops, event rewards, and the crafting system. Strategically combining equipment sets is the key to late-game combat.

UPGRADE SYSTEM

Companion Upgrade System

Progress through each upgrade step -- every stage is critical to your companions' growth

1

Level Up

Use EXP potions to increase base companion level. The level cap increases with your account level (prioritize core characters)

2

Skill Upgrade

Use skill books to enhance skill levels, improving damage multipliers and effect values (prioritize main DPS skills)

3

Awakening / Limit Break

Collect character fragments to break the level cap, unlock new passive skills and appearance changes (critical milestone)

4

Equipment Enhancement

Spend gold and enhancement stones to raise gear levels. Every +5 upgrade has a chance for a super success (prioritize weapons)

5

Set Bonus Activation

Equip multiple pieces from the same set to activate set bonuses. Mix and match different sets for strategic variety

BUILD PATH

Recommended Build Paths

Optimized progression routes for players at different stages

Early Game Build Path

Level main characters to 30 → unlock all skills → equip R-rank sets → push main story to Chapter 2. Focus resources on 1-2 core DPS characters.

Mid Game Build Path

Break main characters to level 50 → upgrade skills to level 5 → collect SR-rank equipment sets → start challenging raids. Build 2 complete teams for different scenarios.

Late Game Build Path

Max level break core characters → max all skills → complete SSR/UR sets → climb arena ladder. Pursuing perfect stat rolls and set combinations becomes the core endgame loop.
