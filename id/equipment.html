Pengenalan Sistem Equipment

Sistem equipment di Star Sailors adalah cara utama buat ningkatin power karakter. Setiap partner bisa dipasangin tiga tipe equipment: Weapon, Armor, dan Accessory. Equipment dengan rarity berbeda punya bonus stats dan set effect yang beda-beda. Equipment bisa didapetin dari dungeon drop, event reward, dan sistem crafting. Kombinasi set equipment yang tepat adalah kunci penting di late game.

UPGRADE SYSTEM

Sistem Upgrade Partner

Ikuti langkah-langkah pengembangan partner, setiap tahap itu krusial

1

Level Up

Pakai EXP Potion buat naikin base level partner. Level cap naik seiring level akun kamu (prioritasin karakter inti dulu)

2

Skill Upgrade

Pakai Skill Book buat naikin level skill, ningkatin damage multiplier dan nilai efek (prioritasin skill DPS utama dulu)

3

Awakening (Breakthrough)

Kumpulin fragment karakter buat nembus level cap, unlock passive skill baru, dan perubahan tampilan (node pengembangan krusial)

4

Equipment Enhancement

Pakai gold dan enhance stone buat naikin level equipment. Setiap kelipatan +5 ada chance super success (prioritasin weapon dulu)

5

Aktivasi Set Effect

Pasang beberapa piece dari set yang sama buat aktifin set effect. Kombinasikan berbagai set secara strategis

BUILD PATH

Rekomendasi Jalur Build

Rencana pengembangan optimal untuk setiap fase permainan

Jalur Build Early Game

Prioritas: naikin karakter inti ke level 30 → unlock semua skill → pakai R set equipment → push cerita utama ke Chapter 2. Fokusin resource ke 1-2 karakter DPS utama.

Jalur Build Mid Game

Karakter inti breakthrough ke level 50 → skill level 5 → kumpulin SR equipment set → mulai challenge Team Raid. Bentuk 2 tim lengkap buat berbagai skenario.

Jalur Build Late Game

Karakter inti full breakthrough → semua skill max level → SSR/UR set complete → push Arena ladder. Ngejar perfect substat dan kombinasi set adalah core loop late game.
