装备系统简介

星际水手的装备系统是提升角色战力的核心途径。每位伙伴可装备武器、防具、饰品三类装备，不同稀有度的装备拥有不同的属性加成和套装效果。装备可通过地下城掉落、活动奖励和锻造系统获取。合理搭配装备套装是后期战斗的关键。

UPGRADE SYSTEM

伙伴升级系统

按步骤完成伙伴养成，每一步都至关重要

1

等级提升

消耗经验药水提升伙伴基础等级，等级上限随账号等级提升而解锁（优先培养核心角色）

2

技能升级

消耗技能书提升技能等级，提高技能伤害倍率和效果数值（优先升级主力输出技能）

3

突破觉醒

收集角色碎片突破等级上限，解锁新的被动技能和外观变化（关键养成节点）

4

装备强化

消耗金币和强化石提升装备等级，每+5级有一次大成功机会（优先强化武器）

5

套装激活

穿戴同一套装的多个部件激活套装效果，策略性搭配不同套装组合

BUILD PATH

养成路线推荐

针对不同阶段的玩家，提供最优养成路线规划

新手期养成路线

优先将主力角色提升至30级 → 解锁全部技能 → 装备R级套装 → 推进主线至第二章。资源集中投入1-2个核心输出角色。

中期养成路线

主力角色突破至50级 → 技能升至5级 → 收集SR级装备套装 → 开始挑战团队副本。组建2支完整队伍应对不同场景。

后期养成路线

核心角色满级突破 → 全技能满级 → SSR/UR套装毕业 → 天梯竞技场冲榜。追求完美词条与套装搭配是后期核心玩法。
